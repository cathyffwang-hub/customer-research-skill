# 客户调研大纲生成 Skill - 安装指引

> 本文档适用于安装「客户调研大纲生成」skill。

---

## ✅ 前置条件

1. 你的电脑已安装 **git**
   - macOS：`brew install git`
   - Ubuntu/Debian：`sudo apt install git`
2. 仓库已公开，任何人都可以 clone

---

## 📦 安装 Skill（仅需一次）

### 第 1 步：克隆仓库

```bash
git clone https://github.com/cathyffwang-hub/customer-research-skill.git ~/.workbuddy/skills/客户调研大纲生成
```

或者使用 SSH（需配置 SSH Key）：
```bash
git clone git@github.com:cathyffwang-hub/customer-research-skill.git ~/.workbuddy/skills/客户调研大纲生成
```

### 第 2 步：注册自动更新（可选）

如果希望每次打开 WorkBuddy 都能自动同步最新版本，可以注册自动更新任务：

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/install.sh
```

看到以下输出说明成功：
```
✅ 检测到仓库：~/.workbuddy/skills/客户调研大纲生成
🔄 正在注册自动更新任务（每 1 分钟检查一次）...
✅ macOS 自动更新任务已注册（launchd）
```

---

## 🔄 自动更新说明

安装完成后，**无需任何操作**：

- 管理员（`cathyfwang`）推送更新后
- 你的电脑**最多 1 分钟内**自动拉取最新版本
- 下次打开 WorkBuddy 即可使用最新版 skill

### 查看更新日志

```bash
cat ~/.workbuddy/skills/客户调研大纲生成/auto_update.log
```

### 手动触发更新（可选）

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/update.sh
```

---

## 🗑️ 卸载自动更新（可选）

如果不需要自动更新，可以运行：

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/uninstall_auto_update.sh
```

## ❓ 常见问题

### Q：克隆时提示 `Permission denied (publickey)`
**A**：如果使用 SSH 克隆，需要先配置 SSH Key。或者使用 HTTPS 克隆（无需配置）：
```bash
git clone https://github.com/cathyffwang-hub/customer-research-skill.git ~/.workbuddy/skills/客户调研大纲生成
```

### Q：安装后 WorkBuddy 找不到 skill
**A**：检查目录是否存在：
```bash
ls ~/.workbuddy/skills/客户调研大纲生成/
```
如果目录存在但仍找不到，请重启 WorkBuddy。

### Q：如何确认自动更新正在运行？
**A**：macOS 用户可以运行：
```bash
launchctl list | grep workbuddy
```
看到 `com.workbuddy.skill.customer-research` 说明任务正在运行。
