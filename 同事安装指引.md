# 客户调研大纲生成 Skill - 安装指引

> 本文档适用于 **Tencent-docx-CSM** 组织成员安装「客户调研大纲生成」skill。

---

## ✅ 前置条件

1. 你已加入 **Tencent-docx-CSM** 组织（GitHub）
2. 你的电脑已安装 **git**
   - macOS：`brew install git`
   - Ubuntu/Debian：`sudo apt install git`
3. 已配置 **SSH Key** 并添加到仓库 Deploy Keys（见下方「配置 SSH Key」）

---

## 🔑 配置 SSH Key（仅需一次）

### 第 1 步：生成 SSH Key

运行以下命令（把邮箱换成你自己的）：

```bash
ssh-keygen -t ed25519 -C "你的邮箱@tencent.com" -f ~/.ssh/id_ed25519 -N ""
```

### 第 2 步：复制公钥

```bash
cat ~/.ssh/id_ed25519.pub
```

复制输出的**整行内容**（以 `ssh-ed25519` 开头），发给 **@cathyfwang（王亚菲）**。

### 第 3 步：等待管理员添加

cathyfwang 把你的公钥添加到仓库 Deploy Keys 后，你会收到通知。

### 第 4 步：测试 SSH 连接

```bash
ssh -T git@github.com
```

看到 `You've successfully authenticated` 就说明配置成功。

---

## 📦 安装 Skill（仅需一次）

### 第 1 步：克隆仓库

```bash
git clone git@github.com:Tencent-docx-CSM/customer-research-skill.git ~/.workbuddy/skills/客户调研大纲生成
```

### 第 2 步：注册自动更新

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/install.sh
```

看到以下输出说明成功：
```
✅ 检测到仓库：~/.workbuddy/skills/客户调研大纲生成
🔄 正在注册自动更新任务（每 1 分钟检查一次）...
✅ macOS 自动更新任务已注册（launchd）
```

---

## 🔄 自动更新说明

安装完成后，**无需任何操作**：

- 管理员（`cathyfwang`）推送更新后
- 你的电脑**最多 1 分钟内**自动拉取最新版本
- 下次打开 WorkBuddy 即可使用最新版 skill

### 查看更新日志

```bash
cat ~/.workbuddy/skills/客户调研大纲生成/auto_update.log
```

### 手动触发更新（可选）

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/update.sh
```

---

## 🗑️ 卸载自动更新（可选）

如果不需要自动更新，可以运行：

```bash
bash ~/.workbuddy/skills/客户调研大纲生成/uninstall_auto_update.sh
```

## ❓ 常见问题

### Q：克隆时提示 `Permission denied (publickey)`
**A**：SSH Key 未配置或未添加到 Deploy Keys。请检查：
1. SSH Key 是否已生成（`cat ~/.ssh/id_ed25519.pub` 是否有输出）
2. 公钥是否已发给 cathyfwang 并等待添加

### Q：安装后 WorkBuddy 找不到 skill
**A**：检查目录是否存在：
```bash
ls ~/.workbuddy/skills/客户调研大纲生成/
```
如果目录存在但仍找不到，请重启 WorkBuddy。

### Q：如何确认自动更新正在运行？
**A**：macOS 用户可以运行：
```bash
launchctl list | grep workbuddy
```
看到 `com.workbuddy.skill.customer-research` 说明任务正在运行。
